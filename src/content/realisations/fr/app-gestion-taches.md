## Le projet

Une application de gestion de tâches complète avec drag-and-drop, collaboration en temps réel et notifications.

## Fonctionnalités

- Tableaux Kanban avec drag-and-drop
- Collaboration en temps réel via WebSocket
- Authentification JWT
- API REST documentée avec OpenAPI

## Stack technique

- **Frontend** : React, TypeScript, Tailwind CSS
- **Backend** : Node.js, Express, PostgreSQL
- **Temps réel** : Socket.io
- **Déploiement** : Docker, GitHub Actions
