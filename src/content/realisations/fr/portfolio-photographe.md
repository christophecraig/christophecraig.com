## Le projet

Création d'un site portfolio pour un photographe professionnel. L'objectif principal était de mettre en valeur les photos avec des temps de chargement minimaux.

## Défis techniques

- **Optimisation des images** : utilisation de formats modernes (WebP, AVIF) avec fallback
- **Lazy loading** intelligent pour les galeries
- **Animations** de transition entre les photos au scroll

## Stack technique

- Astro pour la génération statique
- Tailwind CSS pour le design responsive
- Sharp pour l'optimisation automatique des images

Le site obtient un score Lighthouse de 98/100 en performance.
