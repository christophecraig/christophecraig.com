## The project

Custom CMS built with Strapi (MySQL) to power both the website and the bar's display screens. Personal project made for a friend.

## Features

- Strapi back-office for centralized content management
- Public site built with Vue.js consuming the Strapi API
- Screen display application built with Next.js consuming the Strapi API

## Tech stack

- Strapi (MySQL)
- Vue.js
- Next.js
