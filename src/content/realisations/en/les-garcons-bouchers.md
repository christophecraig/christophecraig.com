## The project

Showcase website with online sales for an artisan butcher shop in Cavalaire-sur-Mer. The site allows customers to order beef ribs and other products online.

The site is currently waiting for content from the client.

## Technical challenges

- Stripe integration for online payments
- Real-time notifications via Telegram for new orders
- Google OAuth authentication for administration

## Tech stack

- Astro
- Tailwind
- Turso
- Hosted on Cloudflare Pages
- Payments with Stripe
- Emails with Resend
- Notifications with Telegram
